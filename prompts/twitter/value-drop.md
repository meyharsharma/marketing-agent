# Value Drop — Text-Only Tweet

## What You're Creating

A 3-4 line text-only tweet for X (Twitter). No images, no hashtags. Pure copy that drives curiosity toward Prompt Optimizer.

## Structure

- **Line 1:** Bold insight, counterintuitive claim, or reframe. Stop the scroll.
- **Line 2:** Expand or pivot. Build on line 1 with a surprising detail.
- **Line 3:** Product connection. Mention Prompt Optimizer with a specific detail (model names, Chrome Extension, one-click, etc.)
- **Line 4 (optional):** CTA with link: https://www.promptoptimizr.com/

Not every tweet needs a CTA. Some should be pure value with no link. Mix it up based on the topic. When the insight is strong enough on its own, skip the link. When it naturally leads to "try this," include it.

## Rules

1. **Under 280 characters total.** This is a hard limit.
2. Each line should be its own sentence. Use line breaks between them.
3. No hashtags. No emojis. No em dashes.
4. No hype words: revolutionary, game-changing, unleash, supercharge, 10x, unlock.
5. No generic AI excitement ("AI is changing everything!").
6. Follow the brand voice: confident, technical but accessible, slightly irreverent.
7. Make the reader feel smart, not sold to.
8. Be specific: name models (ChatGPT, Claude, Gemini), name features (Chrome Extension, one-click), use real numbers when possible.
9. Never start with "Did you know" (that's a different category).
10. No markdown formatting in the output. No bold, no backticks.

> Prompt Optimizer works inside ChatGPT, Claude, and Gemini. One click. Five models.

Good:

Good:

Bad (too salesy):
> Prompt Optimizer is the BEST tool for prompt engineering! Try it now!

Bad (too generic):
> AI is amazing and everyone should use it better.

## Output Format

Output ONLY the tweet text. No frontmatter, no markdown headers, no explanation, no character count. Just the raw 3-4 lines separated by line breaks.
