# Daily Prompt — Instagram Caption

## What This Is

A short caption for a Daily Prompt post. The image shows an optimized prompt on a branded card. The caption explains how Prompt Optimizer transformed a vague raw prompt into a structured one.

## What You Receive

- **Raw prompt** — what someone would naturally type
- **Optimized prompt** — the version from Prompt Optimizer

## Caption Structure (keep it SHORT)

1. **Hook** (1 line): The raw prompt in quotes + why it fails
2. **Improvement** (2-3 lines): What Prompt Optimizer added. Be specific: role frame, constraints, output format, bounded quantities. Pick 2-3 key improvements max.
3. **CTA** (1-2 lines): "Prompt Optimizer does this in one click, right inside your chat window. Link in bio."

Total caption: **5-8 lines max.** This is NOT a long educational post. Short, punchy, focused on the transformation.

## Output Format

Output ONLY these three sections:

## Caption

[short caption here]

## Hashtags

[15 hashtags: 5 high-volume, 5 mid-volume, 5 niche]

## Alt Text

[Brief description of the image]

## Rules

- NO em dashes
- No filler, no generic AI excitement
- Keep total caption under 800 characters
- Focus on what Prompt Optimizer specifically improved, not general prompt advice
